🖋️ Book Studio: As Above, So Below

A sophisticated, AI-augmented workspace designed for authors to move from raw inspiration to finished manuscript. This application blends a minimalist, distraction-free aesthetic with powerful generative AI tools powered by Google's Gemini.

✨ Features

📖 Manuscript Management

Dashboard: Get high-level insights and AI-generated novel premises to spark your next project.

Chapter Drafting: Organize your book into structured chapters.

Distraction-Free Editor: A clean, serif-focused writing environment.

🤖 Gemini AI Integrations

AI Co-Writer: Stuck on a scene? The AI analyzes your current prose and generates the next two paragraphs, matching your unique tone and style.

Prose Polishing: Highlight text to instantly rewrite it—make it more descriptive, increase the suspense, or tighten the pace.

Visualizer (Imagen 4.0): Generate cinematic concept art directly from your scene's text to help nail the atmosphere.

Narrator (Gemini TTS): Listen to your work read aloud in a high-fidelity voice (Kore) to catch rhythm issues and clunky phrasing.

🎭 Creative Tools

Character Workshop: "Interview" your characters. Set up a name and personality, and use the chat interface to talk directly to your character's persona to discover their voice.

Lore Extractor: Paste messy transcripts, voice memos, or research dumps. The AI extracts key characters, locations, and concepts directly into your organized notes.

Brainstorming Engine: Instantly generate plot twists and character quirks to add depth to your story.

🛠️ Technology Stack

Framework: React

Styling: Tailwind CSS (Dark/Light mode support)

Icons: Lucide React

AI Models:

gemini-2.5-flash-preview-09-2025: Text generation, co-authoring, and lore extraction.

imagen-4.0-generate-001: High-quality image generation for scene visualization.

gemini-2.5-flash-preview-tts: Text-to-speech narration.

🚀 Getting Started

Prerequisites

Node.js installed.

A Google Generative AI API Key (set within the apiKey variable in workspace.jsx).

Installation

Clone the repository.

Install dependencies:

npm install lucide-react


Run the development server.

🎨 UI Design Philosophy

The app uses a "Dark Academia" inspired palette:

Typography: Uses a mix of high-legibility Sans-Serif for navigation and classic Serif for the writing experience.

Accents: A signature Gold (#c29b46) for interactive elements.

Atmosphere: Deep slates and off-whites to reduce eye strain during long writing sessions.

📝 Usage Tips

Character Workshop: The more detail you put in the "Personality" field, the better the character roleplay will be.

Lore Extraction: This works best with long, unorganized text—great for dumping a 20-minute voice memo transcript.

AI Continue: Ensure you have at least a few sentences written so the AI can accurately match your style.
